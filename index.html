Electric Vehicle Price Prediction and Data Analysis

About the Project

EVAlytics is a data analytics project that explores Electric Vehicle (EV) data to identify key factors affecting vehicle prices and sales trends. It performs data cleaning, preprocessing, and Exploratory Data Analysis (EDA) to uncover insights and patterns in the EV market.

Project Highlights

  • Comprehensive data preprocessing and cleaning
  • EDA using Python, Pandas, NumPy, Matplotlib & Seaborn
  • Visual insights into EV price, brand, and model trends
  • Foundation for future ML-based EV price prediction
